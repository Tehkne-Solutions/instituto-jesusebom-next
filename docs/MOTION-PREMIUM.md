# Motion Premium — Instituto Jesus é Bom

## Objetivo

Adicionar sensação premium, moderna e institucional às 3 versões de home sem comprometer performance, legibilidade ou confiabilidade.

## Componentes criados

- `components/motion/FadeIn.tsx`
- `components/motion/RevealText.tsx`
- `components/motion/StaggerContainer.tsx`
- `components/motion/StaggerItem.tsx`
- `components/motion/MouseSpotlight.tsx`

## Onde foi aplicado

- Hero das 3 versões com `MouseSpotlight`;
- SectionHeader com reveal suave;
- StatCard, ProjectCard e ComparisonCard com entrada suave;
- CSS global com microinterações premium em cards, botões, tabs, imagens e hero panel.

## Diretrizes mantidas

- Copy oficial preservada.
- Dados oficiais preservados.
- Sem criação de telefone, PIX, CNPJ, parceiros ou depoimentos não confirmados.
- Respeito a `prefers-reduced-motion`.
- Efeitos focados em `opacity` e `transform`, evitando layout shift.
